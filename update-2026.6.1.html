🚀 أبرز التحسينات

  • استعادة أنظف من تنفيذ الأدوات المتقطعة، وارتباطات الجلسات القديمة، وتسليم الوسائط عبر الوكلاء وواجهة CLI
  • استقرار أكبر في القنوات والتسليم عبر الموبايل: Telegram، WhatsApp، iMessage، Slack، Discord، Microsoft Teams، Google Chat، Google Meet، وiOS Talk
  • ربط أعمق لمؤقتات الموفرين والإضافات (retries, OAuth, device-code, تنزيل الوسائط، واستطلاعات المحتوى المولّد) لمنع تعليق التشغيلات
  • تقليل العمل المتكرر في Skills، بيانات الجلسات، حالة البوابة، مراقبي الذاكرة، وكتابة المخازن مع الحفاظ على ثبات الإعدادات
  • تحميل أوضح لـSkills والإضافات عند فشل التحميل أو اللقطات القديمة المعطّلة مع إرشادات استعادة أفضل
  • توسعة Workboard، مظاهر إضافة SecretRef، ترحيل iOS push المستضاف، وتغليف Copilot/Tokenjuice الخارجي
  • ورشة المهارات (Skill Workshop): تجربة Control UI أعمق مع قوائم مقترحات، إجراءات اليوم، تسليم المراجعات، بحث في الملفات، حالات المراجعة، وتوجيه جلسات قابل لإعادة الاستخدام
  • إصلاحات في Control UI: الإرسال يبقى نشطًا أثناء تحميل السجل، بث deltas تدريجيًا، تخطي Markdown أثناء البث، مسح الملحن بعد الإرسال، وقياس زمن أول إخراج
  • دعم MiniMax M3، وإصلاحات Google/Vertex catalog، وتخزين OpenRouter SQLite، وقدرات Copilot Claude 1M، ومحاذاة Foundry reasoning
  • iMessage Monitor وiPhone Plugin Install Ledgers انتقلوا إلى SQLite-backed state لاستعادة أسرع وأقل فحصًا للملفات
  • تحسينات Release/CI/Docker/E2E: حدود للسجلات، اختبارات جاهزية أسرع، استطلاعات حالة، وعمليات workflow أكثر استقرارًا
