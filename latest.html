🎯 أبرز المميزات الجديدة

📱 تسليم تيليجرام بشكل أغنى جديد

تيليجرام الآن يدعم إرسال HTML غني، يحافظ على المسارات الخاصة بالملصقات، يعرض مسودات التقدم ونتائج الأوامر بشكل أوضح، ويتعامل مع جداول HTML بشكل آمن. كما يحافظ على توجيه الإشارات والمعالجات المؤجلة بشكل صحيح.

🔄 استعادة الوكيل بشكل أكثر موثوقية محسّن

تحسينات كبيرة على إعادة المحاولات، النتائج النهائية، الاستخدام بعد الضغط (compaction)، إصلاح تاريخ الجلسات، ومصالحة الردود. الآن الجلسات المتقطعة أو الجزئية تصل لنتائج نهائية مرئية بشكل أفضل.

⚡ تكامل Codex أقوى محسّن

إضافة موافقات تلقائية على الإضافات، توجيه GPT-5.3 Spark OAuth، تنفيذ العقد البعيدة كأداة ديناميكية، وإنهاء أكثر استقراراً لـ app-server.

📦 إضافات مزودي الخدمة المستقلة جديد

حزم المزودين الخارجيين أصبحت الآن إصدارات npm رسمية من الدرجة الأولى. إضافات القنوات المثبتة خارجياً تُحمَّل عند بدء تشغيل Gateway، و StepFun متاح الآن من npm و ClawHub.

💻 عملاء الويب والموبايل أكثر قدرة محسّن

واجهة Control UI أضافت شريط مساحة عمل الجلسات وحالة الإضافات. iOS أضاف أدوات تحكم Apple Watch، و Android يعرض سياق المحادثة.

🔍 بحث ومهارات أكثر فائدة محسّن

بحث Codex المستضاف أصبح متاحاً، مزودي البحث بدون مفاتيح يظلون خيارين متعمدين، وتثبيت مهارات ClawHub يحافظ على مصدر موثّق.

🚀 تعليمات التحديث

💡 ملاحظة: قبل التحديث، يُفضل عمل نسخة احتياطية من إعداداتك الحالية.

الطريقة الأولى: تحديث عبر npm

  1. npm install -g openclaw@latest
  2. openclaw gateway restart
  3. openclaw status

الطريقة الثانية: تحديث عبر Gateway Tool

  1. openclaw gateway update
  2. openclaw gateway restart

التحقق من التثبيت
