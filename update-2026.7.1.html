🎯 أبرز المميزات الجديدة

Control UI

إعادة تصميم واجهة التحكم

تنظيم محادثات أسهل، مهام مباشرة، ضوابط محادثة أوضح، عروض استخدام وتكلفة أفضل، تحميل الملفات، الإقران، الموافقات، وحالة Gateway كلها قريبة من المحادثة.

Onboarding

إعداد أسهل من التثبيت لأول محادثة

إعداد موجّه بخطوات واضحة نحو أول محادثة ناجحة، فحص الاتصالات قبل حفظها، والحفاظ على الخيارات السابقة في حال انقطاع الإعداد.

Mobile & Desktop

تحديثات كبيرة لتطبيقات iOS و Android و macOS

عمل واسع عبر الإعداد، التنقل، المحادثة، الصوت، الصلاحيات، التعريب، الملفات، العمل المجدول، القراءة بدون اتصال، الإرسال في الطابور، استعادة الاتصال، وضوابط الجلسات الأصلية.

Models

دعم GPT-5.6 و Tencent Hy3 و Meta Muse Spark 1.1

توافق GPT-5.6 عبر مسارات OpenAI و Codex المدعومة، إضافة مسار إعداد كامل لـ Tencent Hy3، دعم Meta Model API لـ Muse Spark 1.1، بالإضافة لتوسيع Claude و Ollama و ClawRouter و LongCat.

Coding Agents

تحسينات Codex ووكلاء البرمجة

أمر openclaw attach يمنح Claude Code وصول مؤقت لجلسة محددة، Codex delegation والـ sub-agents يعودون بنتائج متتبعة بشكل أفضل، Copilot يحصل على خيارات مزود أوسع.

Gateway

إصلاح مشكلة حلقات إعادة تشغيل Gateway

البوابات التي تفشل بشكل متكرر الآن تترك مسار إصلاح مستقر بدلاً من إعادة التشغيل اللانهائي.

Automation

تحسينات العمل المجدول والمتصفح عن بُعد والأطراف

العمل المجدول يمكنه الاستيقاظ فقط عند حدوث تغيير، علامات التبويب الموقّعة في المتصفح يمكن إقرانها عن بُعد وحفظ التنزيلات بأمان، أطراف طرفية محمية متاحة على الويب و iOS و Android.

🔄 طريقة التحديث

لتحديث OpenClaw إلى الإصدار الجديد:

npm update -g openclaw

أو للتثبيت الصريح لأحدث إصدار:

npm install -g openclaw@latest

⚠️ ملاحظة: يتطلب Node.js الإصدار 22.22.3+ أو 24.15.0+
